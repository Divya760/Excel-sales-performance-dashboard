# Excel Sales Performance Dashboard

## Project Overview
This project demonstrates how to analyze sales data using Microsoft Excel by creating an interactive Pivot Table Dashboard.

## Features
- Total Revenue KPI
- Sales by Category
- Yearly Sales Trends
- Department-wise Revenue
- Interactive Pivot Tables
- Pivot Charts
- Slicers for filtering

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Dashboard
The dashboard provides a clear visualization of sales performance, making it easier to identify trends and compare revenue across categories and departments.

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Pivot Tables
- Data Visualization
- Dashboard Design
